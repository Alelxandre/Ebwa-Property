Guide de Création de Maison

Étape 1 : Connexion à votre compte

Assurez-vous d'être connecté à votre compte propriétaire. Si vous n'avez pas encore de compte, inscrivez-vous ici.

Étape 2 : Accéder à la section de création de maison

Après vous être connecté, allez dans votre espace personnel et cliquez sur "Créer une maison".

Étape 3 : Remplir le formulaire de création

Remplissez tous les champs requis dans le formulaire de création de maison, y compris :

  • Type de propriété
  • Prix
  • photo(qui sera celle de l'aperçu en générale)
  • ...
Étape 4 : Soumettre le formulaire

Après avoir rempli tous les champs, cliquez sur le bouton "Ajouter Maison" pour soumettre votre maison.

Étape 5 : Publication de la maison

Une fois la maison créée, elle apparaîtra dans votre liste de maisons. Cliquez sur le bouton "Publier ?" pour rendre votre maison visible aux autres utilisateurs.

Étape 6 : Gérer vos maisons

Vous pouvez toujours revenir dans votre espace personnel pour modifier ou supprimer vos maisons existantes.
